# Homework Tracker
Homework Tracker is a daily and weekly task tracker for your characters, heavily inspired by Haldu's War Within Helper.

It covers the stuff that's easy to forget or lose track of across multiple characters — reputations, currencies, rares, weeklies, among other things.

## Commands:

- ` /hw ` — Open config  
- ` /hw toggle ` — Show / hide the tracker
---

## FAQ

**Q: Can I hide or minimize the tracker?**

**A:** Yes! You can minimize the tracker by clicking on the header or hide it using the `/hw toggle` command.

<br>

**Q: How do I show only specific modules in the tracker?**

**A:** You can open the config panel using the `/hw` command and disable the modules you don't want to see from the General tab.

<br>

**Q: Is it possible to resize the tracker?**

**A:** The tracker can be resized by going to the config panel in the Appearance tab and changing either the scale, width or height of the tracker.

<br>

Currently supports Midnight and The War Within.
